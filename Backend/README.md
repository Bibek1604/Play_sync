# TypeScript Express Backend

Quick starter for a TypeScript + Express backend.

Prerequisites
- Install Node.js LTS for Windows: https://nodejs.org/en/download/


# Getting started (PowerShell)

```powershell
# Open PowerShell and run:
node -v
npm -v

# From this project folder:
cd 'c:\Users\bibek\Documents\Developer\Api\Backend'
npm install

# Development (auto-restart on changes)
npm run dev

# Build and run production bundle
npm run build
npm start
```

Notes
- `dev` uses `ts-node-dev` for fast reloads.
- If you don't want the dev server, you can run `npx ts-node src/company/server.ts` instead.
